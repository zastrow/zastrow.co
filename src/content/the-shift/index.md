<h1 class="post-title">The Shift</h1>
<p>The Shift is a project started by my cohorts at <a href="https://seesparkbox.com/foundry/introducing_the_shift">Sparkbox</a>. The idea is that each
month a topic is selected and published on <a href="https://twitter.com/startYourShift">Twitter</a> and via an
<a href="http://seesparkbox.us1.list-manage.com/subscribe?u=c2fcafb7ccc2db34e99075bb1&amp;id=bb05c6c1af">email newsletter</a>. The goal is to encourage
a wide conversation on topics pertinent to the web industry. Then, at the end of the month, each contributor publishes their response to the topic, then shares it on Twitter using the
<a href="https://twitter.com/hashtag/startYourShift?src=hash">#startYourShift</a> hashtag to group the conversation together.</p>
<p>Ever since the project started earlier this year, I have been collecting each link I have seen on publish day. Below is each article that I have collected
to date, organized by month and topic.
</p>
